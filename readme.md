# Noto Sans Only KR (Regular, Black)

 You can use with this code.
 
 ```css
 @import url('https://cdn.rawgit.com/kattergil/Light-Noto-Sans---Webfont/e3184637/styles.css');
 ```

 Regular

```css
p {
    font-family: 'NotoSansKR', sans-serif;
    font-weight: 400;
}
```
 Black

```css
p {
    font-family: 'NotoSansKR', sans-serif;
    font-weight: 900;
}
```
